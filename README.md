# json quick view

JSON查看, 快人一步! 复制内容, 即可自动格式化并展示.

支持 `JSON`, `SQL` 的快速格式展示. 

支持 `windows`, `macOS`, `linux` 平台.

https://github.com/daodao97/json-view/assets/15009280/9329e101-8355-462b-b113-d77ff4239842

## 特点

- 监听剪贴板
- 内容自动格式化
- 窗口自动展示
- 历史记录
